# Lovable Migration Guide

This repository contains comprehensive documentation for migrating and self-hosting your Lovable projects.

## What is Lovable?

Lovable is a platform that allows you to build full-stack applications with AI assistance. You own your code completely and can export, clone, and self-host your projects without any restrictions.

## Documentation

- [Self-Hosting Guide](docs/SELF_HOSTING.md) - Complete guide for migrating your Lovable Cloud project to your own infrastructure

## Key Features

- **Code Ownership**: You completely own all code generated in Lovable
- **GitHub Integration**: Seamlessly sync your code with GitHub
- **Open Source Foundation**: Built on open standards and open-source technologies
- **No Lock-in**: Export and self-host your projects anytime
- **No Proprietary Dependencies**: No hidden frameworks or vendor lock-in

## Getting Started

1. Review the [Self-Hosting Guide](docs/SELF_HOSTING.md) to understand the migration process
2. Set up your GitHub integration in Lovable
3. Choose your hosting platform
4. Follow the migration steps in the documentation
